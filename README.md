# NestJS Scrapper

NestJS Scrapper is a [NestJS](https://github.com/nestjs/nest) playground to learn scrapping.

## Installation

```bash
# Install Dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
